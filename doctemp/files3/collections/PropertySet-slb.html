PropertySet

PropertySet is a set backed by an iterable map with (bytes32, bool) entries.

Insertion, finding, and removing elements are all O(1).

Stores a boolean, and an array index (uint) for each element, in addition to the address.  This is for easy lookup, and for making iteration possible.

Order of insertion is not preserved.
